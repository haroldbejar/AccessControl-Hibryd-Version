# Access Control - Backend API (.NET 9)

Sistema de Control de Acceso para Conjuntos Residenciales - Backend modernizado.

## 🏗️ Arquitectura

- **Clean Architecture** con separación en capas
- **CQRS** con MediatR para casos de uso
- **Repository Pattern** con Unit of Work
- **Entity Framework Core 8** con MySQL

## 🚀 Stack Tecnológico

- .NET 9
- EF Core 9
- MySQL 8.0+
- JWT Authentication
- Swagger/OpenAPI
- Serilog
